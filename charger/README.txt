= Battery charger driver test

== Configuration

Enable [ CXD56xx Configuration -> CXD5247 battery charger ] from SDK
configuration menu.
And also enable this test.

NOTE: This test depends on the board (except Spresense).

== Usage

Just type 'charger' on nsh command line.

  nsh> charger

If test success, show battery charge setting without any error/failure
messages.
